# 資料的取得及保護
團隊成員會從後台伺服器的頻道查看指令使用的資訊。

後台會取得的資訊包含以下:
> 用戶名稱（含 TAG 號）

> 觸發指令的伺服器

> 指令參數（如 `/say` 指令的內容）

> 服務的伺服器名稱及 ID 列表

> 指令使用的時間

HBYC 的後台已經經過基本保護，團隊成員**不會**任意外傳任何資訊，除非為必要或特殊情況才會提供資訊。

必要情況如下:
> 該伺服器管理員要求查看其管理之伺服器 `/say` 指令使用者。

> 違反法律須調查情形。

> NSFW 字眼確認。

**我們不會將個人資料進行商業用途。**
